Script contor vizitatori unici si accesari

Contor 1

Scriptul din "contor1.php" poate fi folosit numai in pagini generate cu PHP.

1. Copiati pe server fisierele "contor1.php" si "contor.txt", in acelasi director in care se afla fisierele dv. PHP in care doriti sa adaugati acest script.
2. In aceste fisiere, adaugati la inceputul lor urmatorul cod (dupa session_start(); - daca aveti cumva aceasta comanda la inceputul fisierului):
    <?php include('contor1.php'); ?>
3. In locul unde doriti sa afisati contorul, adaugati urmatorul cod PHP:
    echo $contor_out;
4. Acordati fisierului "contor.txt" proprietati CHMOD (0766), drepturi depline citire si scriere.

Contor 2

Scriptul din "contor2.php" poate fi folosit atat in pagini generate cu PHP cat si in pagini statice, de tip HTML.
Pentru executarea scriptului este nevoie de un server cu PHP, dar scriptul poate fi apelat, printr-un cod JavaScript si din pagini statice HTML sau pagini exterioare serverului.

1. Copiati pe server fisierele "contor2.php" si "contor.txt"
2. Acordati fisierului "contor.txt" proprietati CHMOD (0766), drepturi depline citire si scriere.
3. In paginile unde doriti sa fie afisat contor-ul adaugati urmatorul cod: 
    
<div id="id_ctr" style="background:#dedefe; width:150px; border:1px solid #bbbbbb; padding:2px;" align="center">
  <script type="text/javascript" src="http://site-ul.dv/contor2.php?pg=nume"></script>
</div>
- site-ul.dv/ este numele site-ului si directorul unde copiati acest script, iar la nume scrieti pentru fiecare pagina unde adaugati acest cod cate un nume diferit
    Exemplu: src="http://www.marplo.net/contor/contor2.php?pg=pagina1"
- Div-ul cu "id="id_ctr"" este cadrul in care e afisat contorul, acesta poate fi personificat grafic de fiecare in functie de pagina.

• Puteti testa si verifica pagina "test.html" (de pe server, ex.: http://localhost/test.html ).
